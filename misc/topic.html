Ransomware in the health industry

Definition

A cyber-attack which usually starts with a victim opening an unknown email that contains a malicious link or attachment where a code is designed to disrupt the victim's computer is being run in the background is Ransomware. The ransomware encrypts the victim's data/system files locking them out and demands payment to unlock (decrypt) to gain access to their data within a set time frame or face the risk of files being deleted thus holding the user ransom (Humayun,2021).

Ransomware is a type of malware designed to restrict access to files and demand a specific condition to regain access to the original files then posing the threat of deleting all files in order to hold the victim ransom. This type of malware works by first infecting the user's computer. One method is embedding malicious code into common file types such as pdf, which you would normally see in email attachments. When opening the file it will open as normal however there is code being run in the background to encrypt all files. This then leads to a splash screen with demands for payment to decrypt and regain access to files within a time frame.

Importace

It is important in a way that it affects the three main keys in cybersecurity.i.e. (vulnerability, availability & integrity) which is an ongoing development affecting people's lives by demanding the payment for decryption keys (Reeve, 2021). With improving technology, software is becoming more accessible to the health industry which could pose a risk for the advanced types of attacks in future. Understanding these types of attacks early on, can give a head start towards preventing them.
